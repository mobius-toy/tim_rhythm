# TIM Rhythm

TIM Rhythm 是一个基于 Web 的振动波形编辑器，专为 TIM 设备设计。它提供了直观的波形创建、编辑和播放功能，支持通过二维码导出波形数据到移动应用。

## 🎵 核心功能

### 🎨 波形编辑器
- **可视化编辑**：直观的波形可视化界面，支持拖拽调整
- **多段波形**：创建复杂的多段波形组合
- **形状变换**：支持线性、缓入、缓出、正弦、保持等多种波形形状
- **实时预览**：实时显示波形效果和播放进度
- **参数调节**：精确控制强度、时长和形状参数

### 🎮 播放控制
- **播放/暂停/停止**：完整的播放控制功能
- **进度控制**：可拖拽的进度条，支持跳转到任意位置
- **时长显示**：精确的时间显示和总时长计算
- **循环播放**：支持单次播放和循环播放模式

### 🎯 预设模板
- **心跳模式**：模拟心跳节奏的振动模式
- **波浪模式**：连续波浪效果的振动模式
- **脉冲模式**：快速脉冲效果的振动模式
- **自定义预设**：支持保存和加载自定义波形预设

### 📤 数据导出
- **二维码导出**：将波形数据压缩并生成二维码
- **即时分享**：通过二维码快速分享波形数据

## 🎯 使用指南

### 基本操作流程
1. **创建波形**：
   - 点击波形区域添加新的波形段
   - 使用右侧编辑器调整参数
   - 选择预设模板快速开始

2. **编辑参数**：
   - **强度**：0-100% 的振动强度
   - **时长**：50ms-5s 的持续时间
   - **形状**：选择波形变换类型

4. **播放控制**：
   - 使用播放按钮开始播放
   - 拖拽进度条跳转位置
   - 调整播放速度和循环模式

5. **导出分享**：
   - 点击"生成二维码"按钮
   - 保存或分享二维码
   - 使用 TIM Nexus 扫描导入

### 波形形状说明
- **线性 (Linear)**：匀速变化，强度线性递增或递减
- **缓入 (Ease In)**：逐渐加速，开始慢后加快
- **缓出 (Ease Out)**：逐渐减速，开始快后减慢
- **正弦 (Sine)**：正弦波变化，平滑的周期性变化

### 预设模板说明
- **心跳模式**：模拟真实心跳的节奏和强度变化，强弱交替跳动，提供节奏感。
- **波浪模式**：连续的波浪效果，强度缓慢起伏，适合放松和按摩体验。
- **脉冲模式**：快速有节奏的脉冲振动，短促高强度，适合提示和提醒。
- **地震模式**：强度由弱到强再逐渐减弱，带有律动感，适合模拟地震般的震动体验。
- **烟花模式**：强度多次快速上升并迅速下降，类似烟花爆发，层次分明富有变化。
- **高强度持续**：从中到高强度，长时间持续输出，专为重度需求设计，如“10秒高”、“40秒高”等。
- **超高频振动**：最大强度恒定输出，特别适合高频需求，如“30秒超高频”、“100秒超高频”。
- **自定义预设**：支持根据用户需求，灵活组合强度、时长、波形形状等自定义保存与加载。
